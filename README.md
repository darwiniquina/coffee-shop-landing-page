# ☕ Coffee Sun Face

A modern coffee brand website built with **React**, **Tailwind CSS**, and **GSAP**.  
A celebration of motion, warmth, and caffeine — powered by scroll-triggered animations and gentle UI transitions.

## 🧠 Tech Stack

- **React** — Component-driven UI
- **Tailwind CSS** — Utility-first styling
- **GSAP + ScrollTrigger** — Smooth animations & scroll interactions
- **AI-generated assets** — Created with Gemini for a cohesive brand aesthetic

---

## 🪄 Features

- Scroll-triggered animations for text and images  
- Responsive layout (desktop & mobile)  
- Parallax effects and motion reveals  
- Embeddable Google Maps for location section  
- Clean, accessible color palette inspired by coffee tones
